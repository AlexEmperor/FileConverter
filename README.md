# \# FileConverter

# 

# WinForms-приложение для конвертации документов

# (Markdown, DOCX, PDF, XLSX).

# 

# \## Возможности

# \- Markdown → PDF / DOCX / HTML / TXT

# \- PDF → DOCX / HTML

# \- XLSX → PDF

# \- Цепочечные конверсии

# 

# \## Используемые библиотеки

# \- Spire.Doc

# \- Spire.PDF

# \- .NET WinForms

# 

# \## Как запустить

# 1\. Открыть решение в Visual Studio

# 2\. Восстановить NuGet

# 3\. Запустить проект

# 

# \## Примечания

# Markdown → HTML / TXT выполняется через промежуточные форматы.
