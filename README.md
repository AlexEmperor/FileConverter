# \# Конвертации в Spire

# 

# \## ❌ Непрямая конвертация

# 

# Чего \*\*напрямую нет\*\*:

# 

# \- Markdown → HTML

# \- Markdown → TXT

# 

# ---

# 

# \## Прямая конвертация

# 

# \### Markdown (Spire.Doc)

# 

# \- Markdown → PDF ✅

# \- Markdown → DOCX ✅

# 

# \### 📄 Word (DOCX) — Spire.Doc

# 

# \- DOCX → PDF ✅

# \- DOCX → HTML ✅

# \- DOCX → TXT ✅

# 

# \### 📄 PDF — Spire.PDF

# 

# \- PDF → DOCX ✅

# \- PDF → HTML ✅

# \- PDF → TXT ❌

# \- PDF → Image (PNG / JPG) ❌

# 

# \### 📊 Excel (XLSX) — Spire.XLS

# 

# \- XLSX → PDF ✅

# \- XLSX → HTML ✅

# \- XLSX → CSV ✅

# 

# \### 📽 PowerPoint (PPTX) — Spire.Presentation

# 

# \- PPTX → PDF ✅

# \- PPTX → HTML ✅

# 

# ---

# 

# Все перечисленные выше прямые конвертации поддерживаются.
